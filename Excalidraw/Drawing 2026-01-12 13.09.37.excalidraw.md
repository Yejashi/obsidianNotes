## Text Elements
EU = SIMD unit 
  - Executes one wavefront per cycle (64)

1 CU = 4 EUs
so:
    Occupancy per CU = Occupancy per EU * 4 ^e2ji1RQk

Each EU has 256 VGPRs per lane (thread)
- A wave (warp) = 64 lanes 

So total registers per EU:
    256 registers * 64 lanes * 2 = 32768 VGPRs per EU ^vBVl84dQ

max_waves_per_EU = (total_VGPRs_per_EU / (VGPPRs_per_thread * 64)) * 2 banks

where:
    total_VGPRs_per_EU = 16384  = 16384 * 2 = 32768 ^fChJl9nG

Resident waves: waves stored in registers, ready to run

Executing wave: waves currently issued this cycle ^f6lE9AF8

Case 1: <= 64 VGPRs
Registers per wave = 64 * 64 = 4096
16384 / 4096 = 4 waves ???? ^Jc53zl2M

Case 4: <= 96 VGPRs
Registers per wave = 96 * 64 = 6144
32768 / 6144 = 5.33-> floor(5.33) = 5 ^pz43NdRb

Each EU has 2 independent VGPR banks
Each bank = 16384 registers

So:
  Bank 0: 16384 VGPRs
  Bank 1: 16384 VGPRs

Each is placed entirely inside one bank ^3TAAf1ok

Case 1: <= 64 VGPRs
Registers per wave = 64 * 64 = 4096
32768 / 4096 = 8 waves ^xL6wEeXp
